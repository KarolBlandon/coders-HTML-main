Formato - h1/h2/h3/h4/h5/h6

H1

La etiqueta H1 también conocida como "etiqueta de título", es la encargada de representar todo el contenido de una página dentro un sitio web.

H2

La etiqueta H2 también conocida como "Etiqueta de subtítulo", es la encargada de dividir el contenido de la página e indicar los componentes del tema propuesto en la etiqueta H1.

H3 - H4

La etiqueta H3 y H4 también conocidas como "Subtemas de Subtítulo", son las responsables de captar las principales ideas que se representan en la etiqueta H2 dentro de una página web.

H5

La etiqueta H5 texto muy pequeño.

H6

La etiqueta H6 texto mucho mas pequeño.

Formato Small

Small: Hace el tamaño del texto una talla más pequeña que el tamaño mínimo de fuente del navegador

Formato Big

Big: hace el tamaño del texto una talla más pequeña que el tamaño mínimo de fuente del navegador

Formato Sub

sub: Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice.

Formato Sup

Sup: define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.

Formato Pre

pre:

Representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo. Los espacios dentro de este elemento también son mostrados como están escritos

Formato abbr

abbr: representa una abreviación o acrónimo.
